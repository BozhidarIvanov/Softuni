We are HTML5 & CSS.

HTML5 is a markup language for describing web documents (web pages).

  • HTML5 stands for Hyper Text Markup Language
  • A markup language is a set of markup tags
  • HTML5 documents are described by HTML5 tags
  • Each HTML5 tag describes different document content


What is CSS?

  • CSS stands for Cascading Style Sheets
  • CSS describes how HTML5 elements are to be displayed on screen, paper, or in other media
  • CSS saves a lot of work. It can control the layout of multiple web pages all at once
  • External stylesheets are stored in CSS files
